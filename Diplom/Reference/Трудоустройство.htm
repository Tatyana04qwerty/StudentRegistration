История трудоустройства всех студентов

Горизонтальное меню в верхей части окна содержит пункт "Списки", который, помимо прочих, включает в себя подпункт "История трудоустройства".

При его выборе открывается окно со спиком всех записей о трудоустройстве студентов.

Для классных руководителей вышеупомянутый подпункт меню содержится в пункте "Классное руководство".

Список содержит ФИО студента, его группу и специальность, место работы, адрес места работы, даты трудоустройства и увольнения, а также дополнительную информацию.

В данном окне имеется возможность отсортировать данные по дате трудоустройства (когда студент был принят на работу) и отобрать записи, которые входят в указанный временной диапазон. Для применения сортировки и фильтрации необходимо нажать на кнопку "Отобрать". Для очистки полей существует кнопка "Сбросить настройки" Поиск отбирает записи по ФИО студента и месту работы автоматически при вводе текста.

Работа с данным списком не предполагает внесения каких-либо изменений. Однако, данный список может быть сохранен в файле Excel по кнопке "Сохранить список". При открытии диалогового окна необходимо выбрать папку, куда будет сохранен файл.

Для классных руководителей доступна история трудоустройства только студентов подчиненных групп.

Редактирование истории трудоустройства выбранного студента

При работе со списками студентов и выпускников по кнопке "Трудоустройство", которая располагается в конце каждой строки над кнопкой "Редактировать", можно перейти к редактированию истории трудоустройства выбранного студента.

В открывшемся окне содержится форма для добавления новой записи о месте работы студента и список истории его трудоустройства. Для добавления записи необходимо заполнить все поля и нажать на кнопку "Добавить". Если студент уже где-то работает, добавить новую запись нельзя. В этом случае необходимо либо нажать на кнопку "Увольнение" над списком (тогда дата увольнения изменится на текущую), либо нажать на кнопку "Редактировать" в конце первой строки списка.

При редактировании записи форма добавления меняется на форму редактирования. После внесения изменений необходимо нажать на кнопку "Сохранить". При нажатии кнопки "Отменить", изменения не будут внесены, а форма изменится на форму добавления. Здесь при нажатии кнопки "Отменить" окно будет закрыто.

При добавлении нового места работы статус студента касательно трудоустройства меняется на "Трудоустроен". При увольнении студента, его статус меняется на "Не трудоустроен".

Особое внимание следует обратить на то, что заранее выбрать дату увольнения нельзя.
