Переключение между списками

При авторизации классного руководителя отображаются подчиненные группы, студенты которых еще обучаются.

Если у преподавателя в подчинении нет ни одной группы, открывшийся список будет пуст. В противном случае, открывшийся список будет отображать студентов первой подчиненной группы (если их несколько). В фильтре групп будут отражены только находщиеся в подчинении.

Для перехода к списку выпускников необходимо выбрать пункт меню "Классное руководство" - "Выпущенные группы".

Для всех списков действуют функции сортировки, поиска и фильтрации, которые применяются по кнопке "Применить". Поиск отбирает записи автоматически при вводе текста.

При выборе подпункта меню "История трудоустройста" открывается соответствующее окно. Подробнее в разделе "Трудоустройство".
