Добавление групп

При выборе пункта "Списки" - "Группы" горизонтального меню в верхей части окна открывается окно со списком всех групп. Данный список содержит номер группы, размер группы (количество студентов в ней), ФИО классного руководителя, специальность и статус.

Специальность группы определяется автоматически, исходя из специальности студентов в ней. Размер группы рассчитывается также автоматически.

При нажатии кнопки "Добавить группу" открывается окно, где необхолимо ввести номер группы. Номер группы должен содержать 5 цифр, где 1 цифра отвечает за номер структурного подразделения, 2 и 3 цифры - за год поступления, 4 цифра - за специальность, 5 - за номер группы. Для сохранения изменений необходимо нажать кнопку "Сохранить".

Если группа с таким номером уже существует, будет выведено сообщение об этом. Группа добавлена не будет.

Если в базе нет группы с введенным номером, она будет добавлена, система выведет сообщение об успешном добавлении. Окно закроется автоматически.

Еще один способ создания группы - перевод студентов в группу, которой еще нет в базе. Подробнее в разделе "Редактирование данных о студенте".

Редактирование групп

Для изменения номера группы необходимо нажать на кнопку "Редактировать" в конце строки выбранной группы.

Откроется окно редактирования. Для отмены изменений необходимо нажать кнопку "Отменить изменения", для сохранения - "Сохранить". Если был введен номер существующей группы, изменения не сохранятся, система выведет соответствующее сообщение. После отмены и успешного сохранения изменений окно закрывается автоматически. Для удаления группы на окне редактирования имеется кнопка "Удалить". Удалить группу можно лишь в том случае, если в ней нет студентов.

Перевод студентов между группами описан в разделе "Редактирование данных о студенте".

Назначение классного руководителя

Над кнопкой "Редактировать" в конце строки списка всех групп находится кнопка "Назначить". При нажатии на нее открывается список преподавателей. Спикок отражает ФИО преподавателя, его контактные данные и количество групп в подчинении. Строка поиска помогает быстро найти преподавателя в списке по его ФИО.

В конце строки выбранного преподавателя необходимо нажать кнопку "Назначить". Если для группы еще не было назначено классное руководство, выводится сообщение о назначении классного руководителя. Иначе выводится сообщение о смене руководителя. После этого окно будет автоматически закрыто.

Количество групп в подчинении у преподавателя рассчитывается автоматически.

Выпуск группы

Для того, чтобы выпустить группу, необходимо вызвать контекстное меню, кликнув правой кнопкой мыши по группе, и выбрать пункт "Выпустить". После подтверждения действия статус группы измениться с "Обучается" на "Выпущена", а студенты группы переместятся в список "Выпускники".
