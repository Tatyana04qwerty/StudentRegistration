Добавление специальностей

При выборе пункта "Списки" - "Специальности" горизонтального меню в верхей части окна открывается окно со списком всех специальностей. Данный список содержит кодовое обозначение специальности, название и аббревиатуру.

При нажатии кнопки "Добавить специальность" открывается окно, где необхолимо ввести ее название и кодовое обозначение. Ввод аббревиатуры специальности обязательным не является.

При нажатии кнопки "Отменить" специальность добавлена в базу не будет, окно закроется автоматически.

Если все поля заполнены корректно и в базе еще нет такой специальности, после нажатия кнопки "Сохранить" система сообщит об упешном сохранении. Окно закроется автоматически.

Редактирование специальностей

Для изменения данных специальности необходимо нажать на кнопку "Редактировать" в конце строки выбранной специальности.

Откроется окно редактирования. Для отмены изменений необходимо нажать кнопку "Отменить изменения", для сохранения - "Сохранить". Если былы введены кодовое обозначение и название существующей специальности, изменения не сохранятся, система выведет соответствующее сообщение. После отмены и успешного сохранения изменений окно закрывается автоматически. Для удаления специальности на окне редактирования имеется кнопка "Удалить". Удалить специальность можно лишь в том случае, если в системе нет студентов, относящихся к ней.
