Добавление пользователя

При выборе пункта "Списки" - "Преподаватели" горизонтального меню в верхей части окна открывается окно со списком всех преподавателей.

В данном списке отрображаются ФИО, контактные данные преподавателя, количество подчиненных групп и роль.

При нажатии кнопки "Добавить преподавателя" открывается окно с формой для заполнения сведений о преподавателе. Обязательные для заполнения поля отмечены красными звездочками.

Если преподаватель уже есть в базе данных, будет выведено сообщение об этом. При заполнении обязательных полей и нажатии кнопки "Далее" необходимо ввести логин и пароль для пользователя или сгенерировать их по кнопке "Сгенерировать".

Если пользователь с введенным логином уже существует, завершить регистрацию нельзя - необходимо придумать или сгенерировать другой логин.

Если все введено верно, то пользователь добавляется в базу и формируется текстовый файл с логином и паролем, который сохраняется на рабочий стол. После сохранения система оповестит пользователя об этом.

Восстановление доступа

Если пользователь забыл пароль, заведующий отделением может сгенерировать для него временный пароль, действующий однократно. После авторизации с использованием временного пароля, он перестает действовать.

Для генерации пароля необходимо в строке выбранного из списка преподавателя нажать на кнопку "Сгенерировать временный пароль". После этого откроется окно для ввода пароля, поле можно заполнить вручную или автоматически по кнопке "Сгенерировать".

При нажатии кнопки "Сохранить" на рабочий стол сохраняется текстовый файл с паролем, который необходимо передать преподавателю.

Когда пользователь авторизируется в системе по временному паролю, ему будет необходимо зайти в аккаунт и запомнить или сменить основной пароль.
